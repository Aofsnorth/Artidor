# Harness Metrics

## Weekly Metrics

- PR count
- Agent-assisted PR count
- CI pass rate
- Failed build count
- Test coverage trend
- Security findings
- Bugs reopened
- Reverts
- Mean time to fix

## Quality Targets

- CI pass rate: > 90%
- Secret findings on main: 0
- Critical dependency vulnerabilities: 0
- Direct pushes to main: 0
- PRs without checklist: 0
- High-risk changes without review: 0
