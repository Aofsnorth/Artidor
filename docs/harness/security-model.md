# Security Model

## Goals

- Protect user media.
- Protect secrets.
- Prevent unsafe AI tool execution.
- Prevent supply-chain attacks.
- Prevent accidental destructive commits.

## Required Controls

- Gitleaks
- Semgrep
- Dependabot
- CodeQL
- Branch protection
- CODEOWNERS
- Least privilege tokens
- Typed AI tools
- MCP call audit log
