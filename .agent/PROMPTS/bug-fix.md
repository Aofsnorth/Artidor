# Bug Fix Prompt

Bug:
<describe bug>

Reproduction:
<steps>

Expected:
<expected result>

Actual:
<actual result>

Fix the bug without creating new bugs.

Rules:
- Reproduce or understand the bug first.
- Find root cause, not just symptom.
- Add regression test when feasible.
- Do not delete failing tests.
- Do not hide errors.
- Do not add workaround unless documented.
- Check edge cases.
- Check security impact.
- Check user data impact.
- Run relevant verification.

Final answer must include:
- Root cause
- Fix explanation
- Regression prevention
- Tests/QA
- Risk
- Rollback
