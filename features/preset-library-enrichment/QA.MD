# QA — Preset Library Enrichment

## Visual distinctiveness checks
- Scan each category grid.
- Adjacent cards should not look like simple clones.
- Preset names should communicate intended style.

## Preview uniqueness checks
- Effects previews show meaningful effect identity.
- Transitions previews show distinct motion/transition type.
- Overlays show visible difference between washes, frames, textures, flashes, vignettes.
- Motion cards communicate motion personality.
- Text cards show distinct typography/layout identity.

## Responsiveness checks
- Asset tabs remain stable on desktop and narrow panels.
- Cards do not clip.
- Labels remain readable.
- No horizontal overflow.

## Interaction checks
- Existing apply/select/preview actions still work.
- Hover/focus states remain usable.
- No broken preset IDs.

## Regression checks
- Existing projects with prior presets still open.
- Tab switching remains responsive.
- No major render-time regression.
- Lint/typecheck/build pass.
