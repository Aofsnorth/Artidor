# Desktop-native Home Screen Parity Plan

## Goal
Bring the native Win32 home screen (`apps/desktop-native/src/ui/welcome.rs`) visually in line with the web homepage (`apps/web/src/app/page.tsx` + `components/landing/hero.tsx`) while keeping the implementation in GDI and preserving the existing hit-test/button state.

## Impact Map
- `apps/desktop-native/src/ui/welcome.rs` — load real PNG assets, draw real logo, headline typography, CTA icons, real editor-preview screenshot, rounded clipping.
- `apps/desktop-native/src/window/mod.rs` — default launch mode changed to `Home` so the screen is visible during development.
- `apps/desktop-native/Cargo.toml` — add `image` (png feature) and `windows-numerics` dependencies.
- `docs/harness/DEPENDENCY_DECISIONS.md` — document dependency rationale.
- `apps/web/public/logos/artidor/logo-uploaded.png` and `apps/web/public/editor-preview.png` — reused as source assets (read-only).

## Design Targets
1. **Header**
   - Real Artidor logo from `logo-uploaded.png` instead of the blue "A" placeholder.
   - Keep nav links and right-side GitHub pill + Open editor button.
2. **Hero lockup**
   - All-serif italic headline ("The video editor that respects your machine.") with "respects" tinted silver to approximate the web gradient accent.
   - Matching eyebrow pill and tagline text.
3. **CTAs**
   - Primary "Open the editor" with a right arrow glyph.
   - Secondary "Star on GitHub" with a GitHub mark, star, and star count.
4. **Preview frame**
   - Browser-chrome frame with traffic lights, "artidor — live preview" title, and "AI editing" badge.
   - Draw the real `editor-preview.png` screenshot instead of a gradient placeholder.
   - Clip the screenshot to the rounded frame.
5. **Stats**
   - Keep the existing 4-column stats strip; values are placeholders matching the web fallback.

## Risks
- The `editor-preview.png` asset may be regenerated by the web build and change content; loading is tolerant of missing files (falls back to placeholder).
- GDI cannot reproduce gradient text, soft glow, or true blur; we approximate with color and silver tint.
- Rounded clipping uses a GDI region; performance is fine for a static home screen.

## QA
- `cargo check` passes.
- `cargo test` passes (existing tests).
- Windows MCP snapshot shows the new logo, headline, CTAs, and real preview screenshot.

## Rollback
- Revert `welcome.rs` to the previous placeholder implementation.
- Remove `image` and `windows-numerics` from `Cargo.toml` and the dependency decision entry.
- Change `window/mod.rs` initial mode back to `Editor` if desired.
